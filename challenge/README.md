# Forbes-Code-Challenge

## Overview
This application takes contents of dictionary.txt and story.txt as inputs and serves an html page
on http://localhost:3000/ containing a list of closest matches for misspelled words in story.txt
from the words found in dictionary.txt.

## Instructions
* Run application by executing `npm i && npm start` from its root directory, it will open a tab
in Google's Chrome with the target html page.
